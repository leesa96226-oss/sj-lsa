# sj-lsa — (주)신정개발 경영지원 대시보드

(주)신정개발의 매출·매입·자금 현황을 보여주는 경영지원 대시보드 프로젝트입니다.

## 구조

- `data/sj_management_data.xlsx` — **데이터 원본(마스터)**. 담당자가 이 엑셀에 입력하고 push하면 대시보드에 반영됩니다.
  - 시트: 안내 / 매출 / 매입 / 거래처 / 프로젝트 / 자금수지 / 설정
  - ⚠️ 시트 이름과 각 시트 1행(제목행)은 대시보드가 읽는 스키마이므로 변경 금지

## 데이터 갱신 절차

1. `data/sj_management_data.xlsx`를 열어 자료 입력 후 저장
2. commit & push → Vercel이 자동 재배포되어 대시보드에 반영

## 배포

- GitHub → Vercel 연동 (main 브랜치 push 시 자동 배포)
