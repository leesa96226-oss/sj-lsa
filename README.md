# sj-lsa — (주)신정개발 경영지원 대시보드

(주)신정개발(무인로봇 · 화학설비크리닝 · 하수도준설)의 매출·매입·프로젝트·자금 현황을 보여주는 경영지원 대시보드입니다.

## 🔒 보안 구조 (중요)

**기업 데이터(엑셀)는 절대 이 저장소에 올라가지 않습니다.**

- 대시보드에 접속한 뒤, **사용자 PC에 있는 엑셀 파일을 직접 선택**하면 브라우저 안(JavaScript)에서만 파싱·계산·표시됩니다.
- 파일은 어떤 서버로도 전송되지 않으며, 파싱 결과는 사용자 브라우저의 localStorage에만 저장됩니다.
- `.gitignore`가 모든 스프레드시트 형식(`*.xlsx`, `*.xls`, `*.csv` 등)과 `data/` 디렉토리를 차단하므로 실수로도 커밋되지 않습니다.

## 데이터 원본

마스터 엑셀: `data/sj_management_data.xlsx` (각자 PC에 보관, 저장소에는 없음)

| 시트 | 내용 |
|---|---|
| 매출 | 청구 거래 — 공급가액·부가세·수금상태·미수금 |
| 매입 | 지급 거래 — 계정과목·프로젝트 원가귀속·미지급금 |
| 거래처 | 매출처/매입처 마스터 |
| 프로젝트 | 현장 단위 계약·진행상태 |
| 자금수지 | 통장 입출금·잔액 |
| 설정 | 드롭다운 목록 원본 |

⚠️ **시트 이름과 각 시트의 1행(제목행)은 대시보드가 읽는 스키마이므로 변경 금지.**

## 사용 방법

1. 대시보드 접속 (Vercel 배포 URL)
2. 마스터 엑셀 파일을 드래그&드롭 (또는 클릭해서 선택)
3. 개요/매출/매입/프로젝트/자금 탭에서 현황 확인
4. 엑셀을 수정했으면 "엑셀 다시 불러오기"로 갱신

## 개발

```bash
npm install
npm run dev   # http://localhost:3000
```

기술 스택: Next.js (App Router) · TypeScript · Tailwind CSS · SheetJS(xlsx) · Recharts

## 배포

GitHub → Vercel 연동. `main` 브랜치에 push하면 자동 배포됩니다.
